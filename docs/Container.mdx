# Container Component

A flexible container component that provides consistent horizontal padding and centered content with a maximum width. This component is built with Tailwind CSS and is fully responsive across all screen sizes.

## Installation

Add the Container component to your shared components:

```tsx
// components/shared/Container.tsx
interface ContainerProps {
readonly children: React.ReactNode;
readonly className?: string;
}

export default function Container({
  children,
  className = "",
}: ContainerProps) {
  return (
    <div className={`container mx-auto px-4 ${className}`}>{children}</div>
  );
}

## Usage Guidelines

### Best Practices

<Callout type="info">
  The Container component is designed to provide consistent spacing and width
  constraints for your content. Here are some recommended practices:
</Callout>

#### Do's ✅

- Use for main content sections that need consistent width/padding
- Nest containers to create layout hierarchies
- Add custom classes via className prop for specific styling needs

// Good - Basic usage

<Container>
  <main>Content here</main>
</Container>

// Good - With custom classes

<Container className="bg-gray-100 py-8">
  <section>Content with background</section>
</Container>

#### Don'ts ❌

<Callout type="warning">
  Avoid these common mistakes when using the Container component:
</Callout>

- Avoid nesting containers unnecessarily
- Don't use for full-width backgrounds/sections

// Bad - Unnecessary nesting

<Container>
  <Container>
    <div>Content</div>
  </Container>
</Container>

// Bad - Full-width section

<Container className="bg-blue-500 w-screen">
  <div>This defeats the container purpose</div>
</Container>

## Technical Details

The Container component utilizes Tailwind CSS classes for its core functionality:

- `container`: Sets a responsive max-width
- `mx-auto`: Centers the container horizontally
- `px-4`: Adds consistent horizontal padding

### Default Breakpoints

The container responds to these Tailwind breakpoints:

| Breakpoint | Width  |
| ---------- | ------ |
| sm         | 640px  |
| md         | 768px  |
| lg         | 1024px |
| xl         | 1280px |
| 2xl        | 1536px |

## Customization

### Custom Widths

You can override the default container width using Tailwind's max-width utilities:

<Container className="max-w-4xl">
  {/* Content with custom max-width */}
</Container>

### Custom Padding

Modify the default padding using Tailwind's padding utilities:

<Container className="px-6 md:px-8">
  {/* Content with custom padding */}
</Container>

## Accessibility

The Container component is a presentational wrapper and doesn't introduce any accessibility concerns. However, ensure your content within the container follows proper accessibility practices:

- Use proper heading hierarchy
- Maintain sufficient color contrast
- Ensure adequate text size and spacing
- Keep content readable at all screen sizes

## Browser Support

The Container component works in all modern browsers that support:

- CSS Flexbox
- CSS Grid
- Modern CSS Box Model
